# Quick Archive

fast and easy YouTube video downloader


## Steps

### Step 1
Get required packages with command below
```
pip -r requirements.txt
```

### Step 2
Run script
```
python app.py "VIDEO_LINK"
```

## Tips

### Specify where to save

Set an output directory
```
        'outtmpl': '/OUTPUT_PATH/%(title)s.%(ext)s',  # 出力場所の指定
```
